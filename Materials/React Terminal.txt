Installation:
Migrate project from 4.0.2 to 4.0.3:
npm install --save --save-exact react-scripts@4.0.3

Install create-react-app: https://www.npmjs.com/package/create-react-app/v/4.0.3
npm i create-react-app

Create react project named(moayad-blog):
npx create-react-app moayad-blog

Download node_modules folder(if not exist):
npm install

Install react router version 5:
npm install react-router-dom@5

Install Redux:
npm install redux --> install redux
npm install redux react-redux --> install react redux

Install Material UI:
npm install @material-ui/core
